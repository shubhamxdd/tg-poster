How It Works — Step by Step### Step 1 — Telegram Setup1. Create a bot via `@BotFather` ? get `BOT_TOKEN`2. Add bot to the private channel as **Admin**3. Register webhook:   ```   https://api.telegram.org/bot<TOKEN>/setWebhook?url=https://yourdomain.com/webhook   ```### Step 2 — Webhook Receives Message- Telegram sends a `POST` request to `/webhook` on every new channel message- Payload contains: message text, media (if any), message ID, timestamp### Step 3 — Claude AI Parses the MessageSince the message format is inconsistent, the raw text is sent to **Claude API** with a structured prompt:```Extract the following fields from this Telegram message and return ONLY valid JSON:{  "title": "",  "type": "movie | series | anime",  "link": "",  "poster": "",  "genre": [],  "year": null,  "language": "",  "description": ""}Message: <raw telegram text here>```Claude handles any format — hashtags, emojis, plain text, structured or unstructured.### Step 4 — Save to MongoDBParsed JSON is validated and inserted as a new document in the `movies` collection.### Step 5 — Frontend Fetches & Displays- `GET /api/movies` returns paginated list with optional filters- Frontend renders a **Netflix-style dark grid** of cards- Each card shows: poster, title, type badge, year, language- Click ? opens streaming link directly---## ??
